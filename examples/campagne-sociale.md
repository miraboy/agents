# Scénario : Campagne Réseaux Sociaux — Lancement de Collection

> Projet comm uniquement — durée estimée : 1 semaine de préparation

## Contexte

Une marque de mode durable lance sa collection printemps.
Objectif : +2000 abonnés Instagram, 500 clics vers la boutique en 2 semaines.
Budget paid : 1500€.

## Flux d'orchestration

**directeur-creatif** → brief orchestrateur-comm

### Phase 1 — Stratégie (2 jours)

1. `brand-strategist` — revue du positionnement pour la collection :
   - Message central de la collection
   - Ton (saisonnier vs institutionnel)
   - Ce qu'on veut que les gens ressentent

2. `stratege-comm` — plan de campagne :
   - Choix des canaux : Instagram (principal) + TikTok + newsletter
   - Calendrier J-14 à J+14
   - Répartition du budget paid (paid social vs influenceurs)
   - KPIs par canal

### Phase 2 — Production de contenu (3 jours, partiellement parallèle)

3. `social-media-manager` — calendrier éditorial détaillé :
   - 14 posts sur 14 jours (fréquence + types + horaires)
   - Captions complètes avec hashtags
   - Stories J0 et J+7
   - Stratégie de collaboration UGC

4. `redacteur-web` — copy pour les ads et la newsletter :
   - 3 variantes de headline pour les Meta Ads
   - Email d'annonce de lancement (J0)
   - Email de récap mi-campagne (J+7)

5. `email-marketer` — séquence newsletter :
   - Email teaser (J-3) : avant-première pour les abonnés
   - Email lancement (J0) : accès prioritaire
   - Email récap (J+7) : best-sellers, témoignages

### Phase 3 — Distribution payante

6. `ads-manager` — campagnes Meta Ads :
   - Campagne notoriété : 500€ (Reels en boosted)
   - Campagne conversion : 800€ (audiences lookalike acheteurs)
   - Campagne retargeting : 200€ (visiteurs boutique)
   - Brief créatif détaillé pour chaque adset

7. `growth-hacker` — plan d'optimisation :
   - A/B test sur les 3 headlines des ads (évaluation J+4)
   - Critères de kill ou scale pour chaque adset
   - Analyse à J+7 et ajustement du budget restant

### Phase 4 — Analyse post-campagne

8. `growth-hacker` + `stratege-comm` — rapport de performance :
   - KPIs atteints vs objectifs
   - Contenu le plus performant
   - Recommandations pour la prochaine campagne

## Handoffs clés

```
brand-strategist → tous les agents de production :
  Fiche de campagne : ton, message central, mots clés visuels

stratege-comm → ads-manager :
  Budget alloué par canal, objectifs CPM et CPC cibles

social-media-manager → redacteur-web :
  Thèmes et sujets des posts pour cohérence cross-canal

redacteur-web → ads-manager :
  3 variantes de headline prêtes pour les créatifs publicitaires

ads-manager → growth-hacker :
  Structure des campagnes + budget dépensé à J+4 pour réévaluation
```

## Escalades potentielles

| Situation | Qui escalade | Vers qui |
|-----------|-------------|----------|
| Positionnement flou pour la collection | stratege-comm | brand-strategist |
| Budget insuffisant pour les objectifs | ads-manager | directeur-creatif (arbitrage) |
| ROAS < 2x après 7 jours | growth-hacker | ads-manager (révision audiences/créatifs) |
| Besoin d'une landing page dédiée | directeur-creatif | chef-equipe (équipe dev) |

## Métriques de succès

- Portée organique Instagram : >50 000 personnes
- Nouveaux abonnés Instagram : +2 000
- Taux d'ouverture emails : >35%
- Clics boutique depuis email : >300
- ROAS paid : >3x
- CPC Meta : <0.80€
